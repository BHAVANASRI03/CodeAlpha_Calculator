
# CodeAlpha Calculator

## Project Overview

This project is a simple and responsive calculator developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Live Demo

https://bhavanasri03.github.io/CodeAlpha_Calculator/

## GitHub Repository

https://github.com/BHAVANASRI03/CodeAlpha_Calculator

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Clear Screen Functionality
* Delete Last Character
* Keyboard Support
* Responsive Design
* Smooth Hover Effects

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

CodeAlpha_Calculator/
├── index.html
├── style.css
└── script.js

## How to Run

1. Clone or download the repository.
2. Open `index.html` in any web browser.
3. Use the calculator to perform arithmetic operations.

## Learning Outcomes

* DOM Manipulation
* Event Handling in JavaScript
* Responsive Web Design
* Frontend Development Basics
